## Special Mobile Behaviors

1. **Search Dropdown**
   - Max height: 60vh on mobile (prevents keyboard overlap)
   - Auto-hides on very small screens when keyboard opens

2. **Pagination**
   - Buttons stack vertically on mobile
   - Full-width on smallest screens
   - Horizontal on tablet+

3. **Filters**
   - One filter per row on mobile
   - 2 per row on tablet
   - 4 per row on desktop

4. **Cards**
   - Smaller poster height on mobile (280px vs 320px)
   - Compact button layout
   - Abbreviated text
